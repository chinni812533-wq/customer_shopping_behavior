# 📊 Data Analytics Project

## 📌 Overview

This project demonstrates an end-to-end data analytics workflow, starting from data loading to delivering business insights through dashboards and reports. It covers data processing using Python, querying databases using SQL, and visualizing insights using Power BI.

The goal of this project is to showcase practical skills in data analysis, data cleaning, and storytelling for decision-making.

---

## 📂 Dataset

* The dataset used in this project contains structured data relevant to business operations.
* It includes features such as customer details, transactions, and product information.
* Data is initially loaded and processed using Python.

---

## 🛠️ Tools & Technologies

* **Python** – Data loading, cleaning, and EDA
* **Pandas, NumPy, Matplotlib, Seaborn** – Data manipulation and visualization
* **SQL (PostgreSQL / MySQL / SQL Server)** – Querying and analysis
* **Power BI** – Dashboard creation and visualization
* **Gamma** – Presentation (PPT) creation

---

## 🔄 Project Steps

### 1. Data Loading

* Imported dataset into Python using Pandas
* Performed initial inspection using `.head()`, `.info()`, `.describe()`

### 2. Exploratory Data Analysis (EDA)

* Analyzed distributions and relationships
* Identified trends, patterns, and anomalies
* Created visualizations for better understanding

### 3. Data Cleaning

* Handled missing values
* Removed duplicates
* Standardized formats (dates, text, categories)
* Fixed inconsistent data entries

### 4. SQL Analysis

* Loaded cleaned data into database
* Wrote SQL queries to extract insights
* Used joins, aggregations, and window functions

### 5. Dashboard Creation

* Connected Power BI to database
* Built interactive dashboards
* Used charts, filters, and KPIs for insights

### 6. Reporting & Presentation

* Documented findings in a structured report
* Created a presentation using Gamma
* Highlighted key insights and business recommendations

## ▶️ How to Run

### Step 1: Clone the Repository

```bash
git clone <your-repo-link>
cd <project-folder>
```

### Step 2: Install Dependencies

```bash
pip install pandas numpy matplotlib seaborn
```

### Step 3: Run Python Scripts

```bash
python eda_cleaning.py
```

### Step 4: Setup Database

* Import cleaned dataset into PostgreSQL / MySQL / SQL Server
* Run SQL scripts from `/sql` folder

### Step 5: Open Power BI Dashboard

* Open `.pbix` file in Power BI
* Refresh data connection

### Step 6: View Report & Presentation

* Open project report (PDF/Doc)
* View Gamma presentation (PPT link/file)
